# Блиц: мой AI-стек

## IDE и AI-плагины
- VS Code + GitHub Copilot / Copilot Chat.
- Для быстрых прототипов иногда использую Cursor.

## Модели под задачи
- Архитектура и рефакторинг: GPT-4/5-класс.
- Быстрые автодополнения: Copilot autocomplete.
- Документация/черновики: Claude или GPT — по контексту задачи.

## MCP
- Файловые и git-интеграции.
- Browser/Playwright MCP для e2e-проверок и скриншотов.
- Интеграции с issue-tracker/knowledge base (если есть в проекте).

## Cursor rules / системные инструкции
Да, использую. Самые полезные:
- тесты и линт обязательны до коммита;
- единый стиль логирования и обработки ошибок;
- без «магических» констант — все в конфиг;
- единая структура PR: summary, risks, migration notes.

## Честно про мой текущий подход
Я активно использую не только ChatGPT/Codex, но и другие нейросети для ускорения работы.
Обычно процесс такой:
1. Через AI быстро набрасываю архитектуру или скрипт.
2. Проверяю руками логику и ищу слабые места.
3. Тестирую в реальной среде (где это возможно).
4. Если работает стабильно — внедряю.

Так я поднимал, например, VPN, бота-трейдера и другие личные проекты: AI помогает с проектированием и черновой реализацией, а финальную ответственность за проверку работоспособности беру на себя.
