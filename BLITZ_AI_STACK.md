# Блиц: мой AI-стек

- **IDE и AI-плагины**: VS Code + GitHub Copilot/Copilot Chat; для быстрых прототипов — Cursor.
- **Модели под задачи**:
  - крупные архитектурные и рефакторинг-задачи — GPT-4/5-класс;
  - быстрые автодополнения в IDE — Copilot autocomplete;
  - генерация документации/шаблонов — Claude/GPT в зависимости от контекста.
- **MCP**:
  - файловые и git-интеграции;
  - browser/playwright MCP для e2e и скриншотов;
  - интеграции с issue-tracker/knowledge base (при наличии в проекте).
- **Cursor rules / системные инструкции**: да, использую.
  Наиболее полезные правила:
  - обязательные тесты/линт до коммита;
  - единый стиль логирования и обработки ошибок;
  - запрет «магических» констант (вынесение в конфиг);
  - явные требования к структуре PR (summary, risks, migration notes).
